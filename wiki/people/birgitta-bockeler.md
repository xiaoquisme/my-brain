## Summary

Software engineer/consultant, author on Martin Fowler's blog. Wrote the influential "Harness Engineering for Coding Agent Users" article (2026-04-02), proposing a systematic framework for controlling AI coding agents.

## Key Points

- Coined/popularized the term "harness engineering" in the context of coding agents
- Framework: feedforward (guides) + feedback (sensors), computational vs inferential execution
- Key insight: harness engineering is an ongoing practice, not one-time configuration
