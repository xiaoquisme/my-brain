## Summary

Engineer at Anthropic. Co-author (with Barry Zhang) of "Building Effective Agents," which established six composable agentic design patterns.

## Key Points

- Co-authored Anthropic's influential guide on agentic system design (2024-12-19)
- Advocates for simplicity: start with the simplest pattern, escalate only when needed
- Emphasizes Agent-Computer Interface (ACI) design — tool documentation as important as prompts
