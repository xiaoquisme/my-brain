## Summary

Addy Osmani 是 Google Chrome 团队的工程经理，专注于 Web 性能和开发者工具。著有多本技术书籍，同时也是 AI 编码 Agent 工程实践的活跃博主。

## Key Facts

- **职位**：Google Chrome 团队工程经理
- **领域**：Web 性能、开发者工具、AI 编码 Agent 工程
- **著作**：《Learning JavaScript Design Patterns》等
- **博客**：addyosmani.com — 撰写大量关于 Agent Harness Engineering、Loop Engineering、Factory Model 等主题的文章

## 关键贡献

在 AI 编码 Agent 领域，Osmani 系统性地阐述了多个关键概念：
- **Agent Harness Engineering**：构建围绕 AI 编码 Agent 的系统性控制
- **Loop Engineering**（[[loop-engineering]]）：设计自动化循环代替手动 prompt
- **Factory Model**（[[factory-model]]）：构建软件的完整系统
- **Cognitive Surrender**：认知投降风险警告
- **Comprehension Debt**：理解力腐烂概念
- **Intent Debt**：意图债务概念
- **Orchestration Tax**：编排税概念
