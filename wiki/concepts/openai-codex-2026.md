## Summary

OpenAI 于 2026年4月16日发布 Codex 重大更新，将一个编码助手扩展为可在电脑上自主操作的全面开发伴侣，服务于每周 300 万开发者。

## Key Points

### 核心能力扩展

- **后台电脑使用**：Codex 可通过视觉、点击和打字来操作电脑上所有应用，多个代理可在 Mac 上并行工作
- **应用内浏览器**：直接在页面上评论提供精确指令，支持前端和游戏开发
- **图像生成**：集成 gpt-image-1.5，在同一工作流中创建产品概念、前端设计原型和游戏素材
- **90+ 新插件**：Atlassian Rovo、CircleCI、CodeRabbit、GitLab Issues、Microsoft Suite、Neon、Remotion、Render、Superpowers 等

### 软件开发生命周期

- 支持处理 GitHub review comments
- 多 terminal tabs
- SSH 连接远程 devboxes（alpha）
- PDF/表格/幻灯片/文档富预览
- 摘要面板追踪代理计划

### 持续性工作

- **自动化扩展**：复用对话线程，保存上下文。自动调度和唤醒继续长期任务
- **记忆（预览）**：记住个人偏好、纠正和耗时收集的信息
- **主动建议**：识别 Google Docs 待处理评论，从 Slack、Notion、代码库拉取上下文，生成优先级行动列表

### 可用性

- 桌面应用用户（已登录 ChatGPT）即日起陆续更新
- 个性化功能将陆续面向 Enterprise、Edu、EU/UK 用户
- macOS 首发的电脑使用功能将扩展到更多地区
